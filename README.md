# Gestor-de-Tareas
Aplicación web sencilla para la gestión de tareas desarrollada con Node.js y Express en el backend, y HTML, CSS y JavaScript en el frontend.

## Tecnologías utilizadas
  - Node.js
  - Express
  - HTML, CSS y JavaScript

## Funcionalidades
  - Agregar nuevas tareas.
  - Marcar tareas como completadas.
  - Eliminar tareas.
  - Interfaz limpia y responsive.

## Instalación

1. Clona el repositorio
   git clone https://github.com/IsmarlinMartinez/Gestor-de-Tareas.git

2. Instala las dependencias
   npm install

3. Inicia el servidor
   npm start

4. Abre en el navegador
   http://localhost:3000
